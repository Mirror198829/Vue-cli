# vue2构建过程
## 安装步骤
### 1.svn下载工程代码
### 2.安装nodeJs
安装nodeJs(已经安装npm),将 npm 的注册表源设置为淘宝镜像
### 3.安装vue命令行工具vue-cli
``` javascript
# 全局安装 vue-cli
$ cnpm install --global vue-cli
```
### 4.安装webpack
`$ cnpm install webpack -g`
### 5.下载依赖
`$ cnpm install`
### 6.启动服务
`$ npm run dev`
浏览器自动打开页面说明运行成功！

<hr/>
### 安装vue命令行工具vue-cli
``` javascript
# 全局安装 vue-cli
$ cnpm install --global vue-cli
# 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project
# 安装依赖，走你
$ cd my-project
$ cnpm install
$ cnpm run dev
```
浏览器自动打开页面说明运行成功！
